Detailed Description

The Controls class provides support to user control.

This class allows to programm the interface for dealing with play/pause/prev/next/volume/elapsed time commands

void Controls::onElapsedChanged ( qint64  value)
slot

onElapsedChanged is signal activated periodically by mainwindow widget

every a new part of the song is played, the stream position changes. This position is comunicated to controls widget using some specific signal.

We need two overloaded function, since the scrollbar widget only deals with integer values and the signal provided by media player offers a 64 bits integer :p
